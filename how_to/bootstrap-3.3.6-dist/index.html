How To: NYT Congress API

This guide will walk you through how to get started using the New York Times' Congress API.

Getting Started

The New York Times' Congress API is available for limited non-comercial use free of charge. Before you start making any requests with the API, you will need to sign up for a NYT account and request an API key. Signing up for a NYT account is free and does not require a NYT subscription. Once you are logged into your account, navigate to the developer API keys page and request a key for the Congress API. There are also a number of other interesting APIs available through the NYT, and you can request a key for each of those from the same page.

Once you have your API key, you'll want to keep it somewhere handy, as you'll need it for any requests you make, but keep it secure. This is not something you want to post publicly (as any requests made with your key are directly connected to your account). In this guide, I will simply use a variable as a placeholder for the key, called "yourAPIKey".
